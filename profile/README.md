## 주요 기능

- 회원가입/로그인 기능
- 지도 마커 기능
- 음식점 상세 보기 기능
- 음식점 및 태그 검색 기능
- 메뉴 선택 기능
- 음식점 스크랩 기능
- 리뷰 작성/수정 기능
